# DrawingApp 

続いてお絵かきアプリです。
画面を指でなぞって絵が描けます。
描いたキャンバスは保存ボタンを押せば、標準の写真アプリに画像として保存できます。
「消しゴム」ボタンをタップすれば描いた線をなぞって消すことができます。

このアプリを通して次の実装が学べます。

 * DragGestureの使い方
 * ジェスチャーの座標情報からPathのViewを作る方法
 * SwiftUIでViewのキャプチャを取る方法
